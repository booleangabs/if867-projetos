Classificador de Uso de máscara por CNN

Luan Cavalcanti, Layse Araújo, Pedro Sarmento

Resumo: Este projeto tem por objetivo de, através de um dateset contendo 853 imagens com diversos n°s de pessoas com máscara, sem máscara e máscara vestida de forma errônea, treinar um modelo CNN para que este possa reconhecer e classificar de forma satisfatória imagens nessas 3 categorias. Fazendo uso do dataset disponibilizado pelo Kaggle, onde cada imagem possui um arquivo xml específico dela contendo as coordenadas dos rostos e a classificação prévia, cortamos cada imagem nos rostos específicos, aumentando o dataset para 4064 imagens, que servem de base para o treinamento (70%), testes (20%) e validação (10%). Como resultado utilizamos 19 épocas para obter uma precisão de 0.96, um recall de 0.95 e um loss de 0.17. Código disponível em https://colab.research.google.com/drive/1OSEPewcB-MjaSf5wBOrKd2_lkxm1aR8v?usp=sharing

Palavras-chave: Rede CNN, Deep Learning, Uso de máscaras, Covid-19

Detecção de vagas disponíveis e previsão de ocupação em estacionamentos em ambientes abertos

Felipe Jun Ting Lin, Gabriel de Melo Evangelista, João Gabriel Vasconcelos, Maria Luísa Lima

Resumo: Encontrar uma vaga livre em estacionamentos de grandes cidades frequentemente se torna exaustivo, consumindo tempo e dinheiro consideráveis. Tecnologias automatizadas capazes de detectar a ocupação de vagas de estacionamento poderiam auxiliar os motoristas que as procuram, informando exatamente onde um local disponível estaria localizado. Além disso, a detecção por imagem permite que câmeras sejam usadas para cobrir grandes áreas, diminuindo custos de, por exemplo, sensores individuais por vaga, que por vezes se encontram danificados, atrapalhando o condutor. Ademais, a previsão da ocupação do local pode auxiliar o motorista a se programar antes de se deslocar, permitindo que seu dia a dia seja facilitado, ou até permitindo que o administrador do estacionamento tome medidas para mitigar a super lotação do espaço. Esse projeto utilizou CNNs, RNNs e Yolov5 para as tarefas de detecção de ocupação e predição.

Palavras-chave: Detecção de ocupação, estacionamento, Previsão de ocupação, Yolo, CNN, RNN

Classificador Multiclasse para Marcas de Tênis

Isaac Ferreira, Walmir Bispo, Wellison Pereira

Resumo: Este projeto tem como objetivo desenvolver uma aplicação de classificação multiclasse de tênis das marcas Nike, Adidas e Converse a partir de técnicas de redes neurais profundas. Para isto, usaremos o dataset disponibilizado na plataforma Kaggle "Nike, Adidas and Converse Shoes Images". Para atingir os objetivos do projeto, a equipe optou por suas soluções de software usando Python como linguagem de programação. A ideia é utilizar as imagens do Dataset como base, pré-processar, utilizar data augmentation para aumentar a base de dados e usa-las como entrada em algoritmos de Redes Neurais Profundas. Nossa ideia e utilizar diferentes arquiteturas de redes convolucionais clássicas e modernas, tais quais VGG16 e a DenseNet, e compará-las através das métricas de loss e acurácia. Para isso, utilizaremos o framework Pytorch aliado ao ambiente de programação Google Colab.
